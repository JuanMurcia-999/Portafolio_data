Web Scraping de colegios de Bogotá

Este proyecto recupera mediante webscraping información específica de todos los colegios de Bogotá directamente de la página del Ministerio de Educación.

Recuperando para un total de 701 colegios datos como:

  • Nombre
  • Dirección
  • Localidad
  • Código DANE
  • Nombre y código de UPZ
  • Si es sede principal
  • Jornadas
